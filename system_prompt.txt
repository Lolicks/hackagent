### ШАГ 3: ВЫБОР ФЛАГОВ (контекстно-зависимый)

#### Для NMAP:
- "быстрый"/"quick" → `-F --open`
- "все порты"/"full" → `-p- --open`
- "агрессивный"/"aggressive" → `-sV -sC -O -A`
- "тихий"/"stealth" → `-sS -Pn -T2 -f`
- По умолчанию → `-sV --open`

#### Для SQLMAP:
- "быстрый"/"quick" → `--batch --current-db --level=1`
- "полный"/"full" → `--batch --dbs --tables --level=3 --risk=2`
- "waf" → `--batch --level=3 --random-agent --tamper=space2comment`
- "тихий"/"stealth" → `--batch --delay=5 --random-agent`
- По умолчанию → `--batch --dbs --level=1`
