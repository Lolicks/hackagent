## 💻 Требования

### Системные требования
- **OS**: Kali Linux / Ubuntu / Debian / Parrot OS (или WSL2 на Windows)
- **Python**: 3.8 или выше
- **Интернет**: Для подключения к API OpenRouter

### Необходимые утилиты (опционально)
```bash
# Установка утилит для Kali Linux
sudo apt update
sudo apt install -y nmap sqlmap gobuster nikto whatweb
```

## 🔧 Установка

### Шаг 1: Клонирование или создание проекта

```bash
# Создаём папку проекта
mkdir ~/deepseek-agent
cd ~/deepseek-agent
```

### Шаг 2: Создание виртуального окружения (venv)

```bash
# Устанавливаем venv если не установлен
sudo apt install -y python3-venv

# Создаём виртуальное окружение
python3 -m venv venv

# Активируем виртуальное окружение
source venv/bin/activate  # Linux/Mac
# или для Windows: venv\Scripts\activate
```

### Шаг 3: Установка зависимостей

```bash
# Устанавливаем необходимые пакеты
pip install --upgrade pip
pip install openai python-dotenv
```

### Шаг 4: Создание файлов проекта

Создайте следующие файлы в папке проекта:

#### 📄 `agent.py` - основной код агента
(скопируйте полный код из предыдущего сообщения)

#### 📄 `.env` - файл с API ключом
```env
OPENROUTER_API_KEY=sk-or-v1-ваш_ключ
```

#### 📄 `system_prompt.txt` - поведение агента
```txt
Ты - автономный ИИ-агент для пентеста Kali Linux. Твоя задача - самостоятельно тестировать цели.

## ТВОЙ АЛГОРИТМ ДЕЙСТВИЙ:
1. Получи цель от пользователя
2. Выбери подходящие утилиты
3. Проанализируй вывод
4. Прими решение о следующих действиях
5. Выдай итоговый отчёт

Будь автономным и профессиональным!
```

## 🔑 Настройка API ключа

### 1. Получение ключа OpenRouter

1. Перейдите на [https://openrouter.ai/keys](https://openrouter.ai/keys)
2. Зарегистрируйтесь или войдите в аккаунт
3. Нажмите **"Create Key"**
4. Скопируйте ключ (начинается с `sk-or-v1-...`)

### 2. Добавление ключа в проект

```bash
# Редактируем файл .env
nano .env
```

Вставьте:
```env
OPENROUTER_API_KEY=sk-or-v1-ваш_скопированный_ключ
```

### 3. Проверка баланса

1. Перейдите на [https://openrouter.ai/credits](https://openrouter.ai/credits)
2. Убедитесь, что есть положительный баланс (для DeepSeek нужно ~$0.50)

## 📖 Использование

### Основные команды

| Команда | Действие |
|---------|----------|
| `протестируй сайт example.com` | Запускает полное тестирование |
| `запусти sqlmap на URL` | Проверка SQL-инъекций |
| `просканируй порты на IP` | Сканирование nmap |
| `проверь директории на URL` | Поиск скрытых путей |
| `/clear` | Очистить экран и историю |
| `/exit` | Выйти из программы |
| `Ctrl+C` | Остановить текущее сканирование |

### Примеры запросов

```bash
# Простое тестирование сайта
┌─[👤] Протестируй сайт example.com

# Сканирование IP адреса
┌─[👤] Просканируй сервер 192.168.1.100

# Проверка SQL-инъекции с известной ошибкой
┌─[👤] У меня есть ошибка SQL на testphp.vulnweb.com/artists.php?artist=1

# Полное тестирование с указанием цели
┌─[👤] Проверь на уязвимости http://testphp.vulnweb.com/artists.php?artist=1
```

## ⚡ Управление сканированием

### Остановка процесса

Если сканирование длится слишком долго:

1. Нажмите **`Ctrl+C`**
2. Агент покажет: `🛑 Сканирование остановлено пользователем`
3. Вы можете ввести новый запрос

### Переключение режимов

- **Автономный режим** - агент сам решает что запускать (по умолчанию)
- **Ручной режим** - вы указываете конкретную утилиту

## ❓ Частые проблемы

### Проблема 1: Ошибка 401 - Invalid API key

**Решение:**
```bash
# Проверьте содержимое .env
cat .env

# Должно быть:
OPENROUTER_API_KEY=sk-or-v1-ваш_ключ

# Пересоздайте ключ на https://openrouter.ai/keys
```

### Проблема 2: Ошибка 402 - Insufficient Balance

**Решение:**
1. Перейдите на [OpenRouter Credits](https://openrouter.ai/credits)
2. Пополните баланс (минимально $5)
3. DeepSeek очень дешёвый - $0.14 за 1M токенов

### Проблема 3: Утилита не найдена

**Решение:**
```bash
# Установите недостающие утилиты
sudo apt install -y nmap sqlmap gobuster nikto whatweb

# Проверьте установку
which nmap sqlmap gobuster nikto whatweb
```

### Проблема 4: Модуль 'dotenv' not found

**Решение:**
```bash
# Активируйте venv и установите зависимости
source venv/bin/activate
pip install python-dotenv openai
```

### Проблема 5: Permission denied при запуске

**Решение:**
```bash
# Дайте права на выполнение
chmod +x agent.py

# Или запускайте через python
python agent.py
```
